# 💳 Payments Intelligence Copilot

An AI-powered analytics platform for payment operations built using **Python**, **Streamlit**, and **Google Gemini**. The application enables payment professionals, business analysts, and operations teams to upload payment datasets, visualize KPIs, and generate AI-powered executive insights in seconds.

---

## 🚀 Project Overview

Payments Intelligence Copilot helps analyze payment datasets such as:

- Transaction Data
- Settlement Reports
- Reconciliation Reports
- Chargeback Data
- Authorization Data
- Merchant Performance
- BIN Analysis
- Acquirer Performance

Instead of manually reviewing thousands of rows in Excel, users can upload a file and receive:

- Interactive dashboards
- AI-generated summaries
- Business insights
- Trend analysis
- Recommendations
- Natural language explanations

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/ataksale/payments-intelligence-copilot.git
```

```bash
cd payments-intelligence-copilot
```

---

## Create Virtual Environment

Windows

```bash
python -m venv .venv
```

Activate

```bash
.venv\Scripts\activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Configure Environment Variables

Create a `.env` file

```text
GOOGLE_API_KEY=YOUR_GOOGLE_GEMINI_API_KEY
```

---

## Run Application

```bash
streamlit run app.py
```
